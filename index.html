Projects

At Base Camp Coding Academy in Water Valley, we have done some projects. Some are terrific, whereas some crashed and burned. These are some of the better projects that I have either worked on by myself, or with others.

GOAT

This project was about who we thought was the GOAT, Greatest Of All Time. We completed this as a sign showing we know and understand basic html and css.

Foundation of Saving Animals

This project was to make a made up charity. We demonstrated how we grew a little in css and html. In this project, we were not forced to use multiple html pages, but for the sake of my mind, I did.

Inventory Manager

This project was a personal project on replit. It helps the management team with inventory. It keeps track of the item, quantity, any notes someone put with it, and the date that it was last ordered.

LOZ Great Plateau Fan Game

This project was to let our directors know that we understand all that we were taught upto that point. It was not required, but if you had finished the rest of the module work the week before, you did the unit project. This was a fun adventure based game that I helped on. Jacen Barefoot and I worked hard on it, and in the end, it worked out fine.

Basic CRUD App

Persistent CRUD App

This is another CRUD app that we had to finish. This one, however, is fixed so that it will remember the password, username, website, and whether or not it was a favorite from past uses. Until it is deleted, it should remember all of the items.
